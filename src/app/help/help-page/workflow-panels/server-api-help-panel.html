Base paths

  • route API routes are served under the same origin as the web app.
  • api Versioned data endpoints use the /api/v1 prefix.
  • folder JBrowse static assets are exposed through /api/jbrowse.

Main endpoint groups

  • storage Genome and taxonomy endpoints return assembly and taxon metadata.
  • hub G4 endpoints return motif rows, distributions, histograms, and downloads.
  • query_stats Research endpoints run microbial environment G4 analyses.
