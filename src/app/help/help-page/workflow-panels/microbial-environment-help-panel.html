Microbial G4 environment research

Compare microbial G4 density with growth temperature or pH across selected strains.

Open research arrow_forward

What to enter

  • device_thermostat Environmental trait: temperature or pH.
  • tune Mode: growth or optimum.
  • filter_alt Optional taxonomy rank and keyword to restrict the strain set.

What you will see

  • monitoring Spearman correlation, p-value, and a scatter plot with a trend line.
  • table_view A strain table with condition value, G4 density, genome size, and taxonomy.

  1. Set the condition

    Choose the environmental condition and the value type to compare.

  2. Choose the strain set

    Use all eligible strains or limit the analysis to selected taxonomy groups.

  3. Run and review results

    Run the analysis, then switch metrics or sort the table as needed.
